FCC Url Shortener

To add a new url pass this to the url above /new/yoururl.

Example: https://url-shortener-freecodecamp.glitch.me/new/www.google.com

Example Output: [
{
"_id": "5a016f15f36d287892020cdf",
"full": "www.google.com",
"short": "https://url-shortener-freecodecamp.glitch.me/0001"
}
]

Just paste the short url in your browser's address bar to get redirected to that website.
